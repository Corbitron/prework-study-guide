HTML

CSS

  • CSS is used to style and layout your web pages.
  • CSS can change your fonts, color, size and spacing of your content and more.
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content insdide an element.
  • link: creates a connection to the external file.
  • rel: attribute specifies the releationship between the current document and the linked document resource
  • href: specifies the locatino (URL) of the external resource

GIT

  • Git is a free opene-source version control system that is the standard for software development
  • git checkout -b feature/(name of feature): will create a new feature
  • git add -A: will add your modification changes in git
  • git status: checks what branch we are currently on
  • checkout -b branch-name: creates a new branch and switches to it
  • pwd displays current working director
  • ls: list of files and folders in the current directory
  • cd: navigates between directories in the pasth
  • mkdir: creates a new directory
  • touch: creates a file inside a directory
  • start: opens a file

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function name is followed by ().
  • The block code is written inside {}.
  • Strings need to be contained in "".
  • const: Keyword that allows us to create variables thta can't be reassigned
  • Each line of code in Javascript end in a (;).
  • Javascript is a programming language that adds interactivity to web pages and apps
